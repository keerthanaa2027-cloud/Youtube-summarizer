# 🎥 YouTube AI Video Summarizer

> Stop watching. Start knowing.

**[▶ Live Demo](https://youtube-summarizer-2hy4.vercel.app)** | **[GitHub](https://github.com/keerthanaa2027-cloud/Youtube-summarizer)**

---

## What is this?

Paste any YouTube link → get AI-powered summary, key points, timestamps, and even **chat with the video** — in seconds.

Built for students, professionals, and anyone who values their time.

---

## Features

| Feature | Description |
|---|---|
| 🤖 AI Summary | LLaMA 3.3 70B generates clean summaries |
| 📌 Key Points | Auto-extracted bullet points |
| ⏱️ Timestamps | AI detects important moments |
| 💬 Chat with Video | Ask anything about the video |
| 🖼️ Image Search | Upload photo → find related videos |
| 🌐 Multi-Language | 8 languages supported |
| 📄 PDF Export | Download summary as PDF |
| 🔗 Share | Generate shareable summary links |
| 🔐 Auth | JWT login/register system |
| 📋 History | All past summaries saved |

---

## Tech Stack

```
Frontend  →  React.js
Backend   →  Python Flask
AI Model  →  LLaMA 3.3 70B via Groq API
Database  →  MongoDB Atlas
Auth      →  JWT + bcrypt
Deploy    →  Vercel + Hugging Face Spaces
```

---

## Run Locally

```bash
# Backend
cd backend
pip install -r requirements.txt
python app.py

# Frontend
cd frontend
npm install
npm start
```
